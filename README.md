# Test_Task_Effective_Mobile

# Инструкция по настройке проекта

Данное руководство описывает шаги по локальной настройке проекта.

## Предварительные требования

Убедитесь, что на вашей системе установлен `git`. Если нет — выполните установку.

## Шаги установки

1. **Установите Git** (если ещё не установлен):

   ```bash
   sudo apt update
   sudo apt install git -y
   ```

2. **Скопируйте репозиторий** на свой компьютер:

   ```bash
   git clone https://github.com/Hantry/Test_Task_Effective_Mobile.git
   ```

3. **Перейдите в директорию проекта**:

   ```bash
   cd Test_Task_Effective_Mobile
   ```

4. **Сделайте скрипт setup.sh исполняемым и запустите его**:

   ```bash
   sudo chmod +x setup.sh
   sudo ./setup.sh
   ```

## Примечания

- Убедитесь, что у вас есть достаточные права для выполнения команд с `sudo`.
- Если в процессе настройки возникают проблемы, проверьте:
  - подключение к интернету;
  - правильность URL репозитория.
- Убедитесь, что ваша система соответствует всем зависимостям, требуемым проектом (подробности можно найти в документации проекта, если она доступна).
